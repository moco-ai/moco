# Alpaca 自律型株式トレーディングシステム 技術仕様書\n\n## 1. プロジェクト概要\nAlpaca Trading API (`alpaca-py`) を活用し、米国株式市場（Paper Trading環境推奨）において自律的に売買を行うトレーディングシステムを構築します。\n\n---\n\n## 2. 技術スタック\n- **Language**: Python 3.10+\n- **Core Library**: `alpaca-py` (v0.30.0以降)\n- **Data Analysis**: `pandas`, `pandas-ta`\n- **Utilities**: `python-dotenv`, `loguru`\n\n---\n\n## 3. クラス構成\n- **ClientWrapper**: 認証・APIクライアント管理\n- **DataProvider**: 最新Barデータ取得\n- **StrategyBase**: シグナル判定抽象クラス\n- **RiskManager**: 2%リスク制限、ブラケット設定\n- **Executor**: 注文の執行管理\n\n---\n\n## 4. 実行フロー (Main Loop)\n1. 初期化\n2. データ取得\n3. 戦略判定\n4. 注文執行\n5. スリープ\n
