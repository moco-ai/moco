# Alpaca 自律型株式トレーディングシステム 技術仕様書\n\n## 1. プロジェクト概要\nAlpaca Trading API (`alpaca-py`) を活用し、米国株式市場（Paper Trading環境推奨）において自律的に売買を行うトレーディングシステムを構築します。\n\n---\n\n## 2. 技術スタック\n- Python 3.10+\n- alpaca-py (最新リクエストモデル使用)\n- pandas, loguru\n\n---\n\n## 3. クラス構成\n- **ClientWrapper**: 認証管理\n- **DataProvider**: Barsデータ取得\n- **StrategyBase**: シグナル判定\n- **RiskManager**: 2%リスク制限、ブラケット注文算出\n- **Executor**: 注文執行\n\n---\n\n## 4. 実行フロー (Main Loop)\n1. 初期化 -> 2. データ取得 -> 3. 判定 -> 4. 数量計算 -> 5. 執行 -> 6. 待機\n
